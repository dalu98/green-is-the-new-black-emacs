Emacs Green Is The New Black - Theme

Green Is The New Black is a blackened green and minimalist theme for Emacs. With coherent hight-light and 256color terminals support.

Screenshots

Supported Modes

  • company
  • flycheck
  • ido
  • and many others coming

Installation

You should install from MELPA

M-x package-install RET green-is-the-new-black-theme

Add at your emacs.d/init.el

(load-theme 'green-is-the-new-black t)
